## 1. What the site is

Virtual Teammate is a **staffing agency for virtual assistants** — medical, dental, and business/admin VAs. The marketing site's job is to:

1. Convince **businesses/practices** to hire a VA (lead generation → "Book a Demo" / "Book a Strategy Session").
2. Recruit **virtual assistants** to join the talent pool (applicant registration).
3. Showcase the **talent pool** (browsable VA profiles).
4. Educate via **case studies, courses, lead magnets, and a blog**.

The public marketing site sits in front of a gated **member portal** (clients, VAs, CSMs) — see the backend doc.

## 5. HubSpot — the marketing engine

HubSpot is the CRM/marketing hub behind everything. Two integration layers:

1. **LeadIn** (official HubSpot plugin, active) — drops the HubSpot **tracking script** site-wide, captures form submissions, and pushes activity into the HubSpot CRM. Connected via OAuth (`leadin_portalId`, `leadin_refresh_token`). This is what powers HubSpot's own forms/popups/chat and visitor analytics.
2. **`vt-hubspot-user-sync`** (custom plugin) — the operational sync: turns HubSpot **contacts** (lead status "Virtual Teammate") into public **VA profile pages**, and HubSpot **companies** (lead status "Client - Active") into portal client accounts. See backend doc for the full state machine.

**Marketing takeaway:** a VA's public profile page is a direct projection of their HubSpot contact record. To change what shows publicly, you edit HubSpot (or the ACF fields), then the sync regenerates the page — you do not hand-edit profile pages.

## 10. Quick "where do I edit X" cheat sheet

| I want to change… | Where |
|---|---|
| A marketing page's layout/copy | Edit the page in **Elementor** |
| Header / footer / global template | **Elementor → Templates → Theme Builder** |
| A lead form or where its email goes | **WPForms → the form → Notifications** |
| A VA's public profile | **HubSpot** contact record (then re-run the sync) → or the ACF fields on the `vt-list-by-category` post |
| SEO title/meta for a page | **Yoast SEO** box on that page |
| Nav items / who sees them | **Appearance → Menus** + **If Menu** rules |
| Tracking / HubSpot connection | **LeadIn** plugin settings |
| Site-wide custom code/shortcodes | **WPCode** snippets (see backend doc) |
